## 一些方法
### 字符串长度
```javascript
let browerType="mozilla"
browerType.length
```
### 检索特定字符串字符
第一个字符：browerType[0]

### 查找子字符串并提取
browerTyper.indexOf("zilla")
结果：2

browserType.indexOf("vanilla");
结果：-1

**提取**
browerType.slice(0,3)
结果：moz
提取从第一个位置开始，直到但不包括最后一个位置

browerType.slice(2)
结果：zilla
没有包含第二个参数，返回的子字符串是字符串中的所有剩余字符
